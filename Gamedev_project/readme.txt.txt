На основе полученных данных в курсе "Аналитик данных" школы Яндекс.Практикум выполнен буткемп-проект по одной из предложенных областей:
- банки;
- ритейл;
- игры;
- мобильные приложения;
- e-commerce.
Проект выполнен по выбору в области Gamedev.
Работа выполнена на python c приложением презентации в форматах PP и PDF
Дополнительно в рамках задания выполнен дашборд в Tableau (доступен онлайн по ссылке https://public.tableau.com/views/AdvsSources/Dashboard1?:language=en-US&publish=yes&:display_count=n&:origin=viz_share_link )
Использовались библиотеки python: pandas, numpy, pyplot, scipy.stats, datetime, seaborn, os
