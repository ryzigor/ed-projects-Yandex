Проект представляет собой анализ результатов AB тестирования условного интернет-магазина.
Тест с большим количеством "шума" в данных и неоднозначными выводами, основанными на допущениях.
Техзадание поставлено школой Яндекс.Практикум в рамках обучения по профессии аналитик данных. 
Использовались библиотеки: pandas, numpy, matplotlib, scipy.stats, seaborn
