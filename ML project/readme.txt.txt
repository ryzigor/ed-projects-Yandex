Проект представляет собой аналитическое исследование взаимодействия фитнес-центра с клиентами с применением методов машинного обучения.
Работа выполнена на python c использованием библиотек:
pandas, sklearn, numpy, pyplot, scipy.cluster, datetime, seaborn, os
Использованы модели машинного обучения:
- бинарной классификации - логистическая регрессия и случайный лес
- кластеризация алгоритмом K-Means
